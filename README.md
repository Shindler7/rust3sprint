# Блог-платформа

Полноценная система блога, которая объединяет технологии: веб-сервер с HTTP 
и gRPC API, клиентскую библиотеку, CLI-инструмент и WASM-фронтенд. Система 
включает регистрацию и аутентификацию пользователей через JWT-токены, что 
позволяет защитить операции создания, редактирования и удаления постов.

Учебный проект. "Яндекс Практикум", "Rust для действующих разработчиков", 2026.
